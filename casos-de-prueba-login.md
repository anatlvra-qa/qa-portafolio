# Caso de Prueba – Login Correcto

**ID:** CP-001  
**Módulo:** Autenticación  
**Prioridad:** Alta  

## Precondiciones
- Usuario registrado en el sistema.

## Pasos
1. Ingresar al sitio.
2. Hacer clic en “Iniciar Sesión”.
3. Escribir email y contraseña válidos.
4. Presionar el botón “Ingresar”.

## Resultado esperado
El sistema debe permitir el acceso y mostrar el dashboard del usuario.
